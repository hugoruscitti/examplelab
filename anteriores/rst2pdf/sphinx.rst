Sphinx - Tarjeta de referencia
==============================

Comandos principales
--------------------

>>> sphinx-quickstart
>>> make html


Sintáxis
--------

Niveles de titulos::

    *****            (capítulo)
    #####            (sección)
    =====            (titulo)
    -----            (subtitulo)
    ^^^^^            (subsubtitulo)

Atributos de fuente::
    
    *cursiva*
    **negrita**
    ``codigo``

Listas::

    * item de lista.
    * mas items.
        * Elemento interno de la lista.

    1. item de lista enumerado.
    2. segundo elemento.
    #. tercer item de la lista.


Definiciones::

    Elemento
        Descripción detallada.


Ejemplos:

Deben comenzar con dos puntos seguidos, y lo siguiente debe
estar identado, por ejemplo:

Veamos un ejemplo::
            
    import pygame
